# Digital Signal Processing Project: Audio Interference Removal

## Project Description
This MATLAB project demonstrates digital signal processing techniques to analyze and remove high-frequency sinusoidal interference (15.2 kHz) from an audio signal. The implementation includes:

1. **Spectral Analysis** using Welch's method
2. **Digital Filter Design** comparing IIR and FIR approaches
3. **Performance Evaluation** of different filter types
